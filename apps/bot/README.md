# Telegram-бот

Здесь будет код Telegram-бота для «М7 платформа». Бот отвечает за авторизацию по @username + одноразовому коду, отправляет уведомления (матчи, отклики, модерация, платежи) и предоставляет быстрые действия, например кнопку «открыть Mini App» или переход в чат с другим пользователем. Бот интегрируется с backend’ом и базой данных.
