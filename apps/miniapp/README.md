# Mini App (Telegram WebApp)

Здесь будет находиться Telegram Mini App для проекта «М7 платформа». В приложении планируется реализовать все ключевые разделы: знакомства, маркет (товары и услуги), жильё и работа. Фронтенд предполагается на Next.js (App Router) и TypeScript, работающий внутри Telegram WebApp с авторизацией через @username + код, полученный от бота.
